# PlantUML Themes Catalog

Benvenuto nel catalogo dei temi personalizzati PlantUML! Questo repository raccoglie vari temi che possono essere importati direttamente nei tuoi diagrammi PlantUML tramite GitHub.

## Come utilizzare i temi

Aggiungi semplicemente la direttiva `!include` all'inizio del tuo diagramma PlantUML seguito dall'indirizzo raw del tema su GitHub.

---

### Theme Modern Soft

Un tema chiaro ed elegante, ispirato alle linee guida per l'accessibilità, con uno sfondo leggermente azzurro (#F4F8FF), l'ideale per presentazioni e documentazione formale e istituzionale.

**Come includerlo nel tuo diagramma:**

```plantuml
!include https://raw.githubusercontent.com/vcherney88/themes_pantuml/main/themes/theme-plunt-modern-soft.puml
```
